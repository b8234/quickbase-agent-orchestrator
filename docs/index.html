Quickbase Agent Orchestrator

Serverless AI workflow enabling conversational access to Quickbase data

Overview

A lightweight reference architecture demonstrating how to combine Amazon Bedrock, AWS Lambda, API Gateway, S3, Transcribe, and Quickbase to build a distributed, event-driven, conversational workflow system.
