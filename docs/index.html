Serverless, Multi-Stage AI Orchestration System for Conversational Access to Quickbase Data

The Quickbase Agent Orchestrator is a distributed AI workflow system that enables users to query, summarize, and export Quickbase data using natural language or voice. It combines Amazon Bedrock, AWS Lambda, API Gateway, S3, Transcribe, Slack, and Streamlit to create an end-to-end, enterprise-ready orchestration pipeline.

  • Enterprise workflow patterns including exports, Slack alerts, and validation

âš™ï¸ System Components

Streamlit Frontend

  • Supports both voice and text input
  • Displays responses, tables, and export links
  • Communicates with backend via API Gateway

Lambda Frontend

  • Handles HTTP requests from the UI
  • Transcribes audio using Amazon Transcribe
  • Invokes Bedrock Agent

Bedrock Agent

  • Routes user intent to backend Lambda via action groups
  • Coordinates multi-step reasoning

Lambda Backend

  • Queries Quickbase records
  • Enforces field allowlist governance
  • Formats and summarizes data
  • Exports CSV/JSON files to S3
  • Sends Slack notifications
